# Indian Data Job Market Dashboard

Power BI and Python project analyzing hiring trends, skill demand, salary patterns, and career opportunities in the Indian Data Analytics job market.

## Overview
This project explores job postings from the Indian data industry to understand:

* Which roles are hiring the most
* Which skills are most in demand
* Which skills command higher salaries
* Experience-level hiring trends
* Geographic distribution of opportunities

Python was used for data analysis and feature creation, while Power BI was used for dashboard development and visualization.

## Project Workflow

### Data Exploration

The dataset was reviewed and analyzed to understand:

* Job titles
* Salary information
* Experience levels
* Locations
* Required skills

The dataset was already largely structured and did not require significant cleaning.

### Feature Creation

Additional datasets were generated for analysis:

#### data_roles.csv

Role-level dataset used for job role analysis.

#### skill_job_map.csv

Mapping table created by splitting multiple skills from each job posting into individual records.


#### skill_analysis.csv

Aggregated skill-level metrics including:

* Demand Count
* Average Salary
* Opportunity Score

## Dashboard 1: Indian Data Job Market Dashboard

### KPIs

* Total Jobs: 853
* Average Salary: 20.6 LPA
* Maximum Salary: 94.8 LPA
* Cities Covered: 17

### Visualizations

* Top Job Roles
* Top Hiring Cities
* Jobs by Experience Level
* Most In-Demand Skills

### Key Insights

* Data Analyst is the most common role in the dataset.
* Data Scientist and Business Analyst positions show strong demand.
* SQL is the most frequently requested skill.
* Mid-level professionals (3–6 years) have the highest number of job opportunities.
* Remote hiring accounts for a significant share of listings.

---

## Dashboard 2: Skill Intelligence Dashboard

### Visualizations

* Highest Paying Skills
* Most In-Demand Skills
* Best Opportunity Skills
* Skill Demand vs Salary Analysis

### Key Insights

* SQL has the highest overall demand.
* Python and Power BI remain highly sought-after skills.
* Machine Learning, NLP, TensorFlow, and Scikit-Learn show higher salary potential.
* Python offers a strong balance between demand and compensation.
* Advanced AI/ML skills generally provide higher salaries but appear in fewer postings compared to foundational skills.
